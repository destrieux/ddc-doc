    Téléchargement des extensions¶

    L'extension don_corps nécessite le téléchargement préalable des extensions suivantes :

    Extension Rôle
    de.systopia.civioffice Génère des documents Word et pdf depuis des modèles Word
    org.civicoop.civirules Déclenche des actions si certaines règles sont réunies
    org.civicoop.emailapi Envoie des courriels lorsque certaines règles sont vérifiées
    org.civicoop.civiruleslogger Logue les actions déclenchées par les règles
    org.civicoop.documents Gère les documents téléchargés (mini GED)
    org.civicrm.afform_admin Génère les formulaires de recherche à partir de requêtes enregistrées
    (déjà téléchargée)
    org.civicrm.contactlayout Gère les grilles d'affichage des contacts
    de.systopia.eventmessages Envoi les courriels relatifs aux évènements (cérémonies...)
    sktokens Crée des tokens depuis la base de données, utilisables dans les messages

    Pour le moment l'extension FrenchCodesPostaux qui utilise l'API de la poste pour compléter les adresses est incompatible avec org.civicrm.contactlayout

    Téléchargement des extensions¶

    • Vous pouvez télécharger la dernière version de chacune des extensions en suivant les liens du tableau ci-dessus.
      Chaque extension doit être téléchargée et dézippée dans le dossier :
      /var/www/html/ddctest/wp-content/plugins/civicrm/civicrm/ext/
    • Vous pouvez aussi utiliser une archive contenant les extensions nécessaires en une fois
      L'archive doit être téléchargée et dézippée dans le dossier :
      /var/www/html/ddctest/wp-content/plugins/civicrm/civicrm/ext/

    Il n'est pas nécessaire d'installer les extensions à ce stade, mais seulement de les télécharger et de les déposer dans le dossier ad hoc;
    Elles s'installeront en même temps que l'extension don_corps.
