# Cérémonies d'hommage
Une cérémonie en hommage aux donneurs et à leurs proches doit être organisée annullement.

Ce processus comporte 3 étapes :

* Créer un *modèle de cérémonie* qui contient les informations communes aux cérémonies (lieu..). A ne faire qu'une fois.
* Utiliser ce modèle pour *créer une cérémonie*. A faire pour chaque cérémonie.
* *Rattacher les PAQPF* à la cérémonie créée. A faire pour chaque décès.
