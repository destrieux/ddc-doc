# Lister les participants à une cérémonie
Allez à : **Cérémonies > Statuts des PAQPF pour cérémonies**.

Les PAQPF des défunts décédés dans les 90 jours précédents (paramétrable) s'affichent.<br>

Vous pouvez filtrer par Date ou nom de cérémonie ou par statut de participant, pour ne garder par exemple que les participants confirmés d'une cérémonie donnée.<br>

Si vous souhaitez télécharger un tableau pour l'imprimer : **Action > Download Spreadsheet**<br>
Choisir format xlsx.
