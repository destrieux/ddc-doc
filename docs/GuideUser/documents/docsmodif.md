# Comment modifier un modèle de document ?
Votre administrateur a dû vous indiquer l'espace disque où sont situés les modèles de documents.

Vous pouvez les modifier comme des documents Word habituels.

Vous pouvez insérer des **tokens** reprenant des informations de la base de données.<br>

Pour lister les tokens disponibles **CiviOffice > Available Tokens**.
