# In memoriam

Les donneurs expriment leur souhait ou non d'affichage de leur nom sur la stèle ou un site web. 

Pour lister les donneurs souhaitant un affichage et décédés sur une période donnée,<br>
allez à **Cérémonies > In memoriam**.

Vous pouvez sélectionner une période pour la date de décès.

Pour exporter : **Actions > Download Spreadsheet > Format xlsx**.
