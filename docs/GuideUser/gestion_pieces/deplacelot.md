# Déplacement de pièces en lot
Il est possible de déplacer de nombreuses pièces ou corps en une seule opération, par la création d'une activité de type *Déplacer un lot de pièces/corps*.

* Ouvrez la fiche contact **du lieu vers lequel vous souhaitez déplacer le lot**,
> Attention à bien ouvrir la fiche contact du **lieu cible** ! 
* **Action > Déplacer un lot de pièces/corps**,
* Entrez la liste de pièces dans le champ *Détails*, **Enregistre**.

Les pièces et corps sont déplacés vers le lieu cible, c'est à dire que le champ *localisation* des utilisations est modifié pour chacun.

Si une des pièces a été préalablement détruite, elle est recrée dans ce local de conservation.

Si un déplacement est demandé pour une pièce inexistante dans la base, une alerte est produite.

Dans tous les cas un log des déplacements est inscrit dans le champ *Détails* de l'utilisation.
