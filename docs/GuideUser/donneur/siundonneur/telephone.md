# Demande des informations par téléphone ou lors d'un rendez-vous
Vous pouvez conserver une trace de cet appel ou de ce rendez-vous en ajoutant une activité : depuis la fiche du Donneur, **Action > Enregistrer Activité**. 

* Choisir le type d'activité : *Réunion, Appel téléphonique*.<br>
* Renseignez les champs ; le champ **Assigné à** permet de transférer une copie du message à un contact, par exemple un de vos collègues du centre, qui devra recontacter le donneur.
* Une activité supplémentaire apparait dans l'onglet **Activités** de la fiche Contact.
