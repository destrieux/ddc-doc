# Présentation de la fiche Donneur
La fiche donneur comporte plusieurs onglets :

![Onglets fiche contact](../../../img/Donneurs_Onglets.png)

* **Résumé** : c'est la partie la plus complexe de la fiche qui contient les informations du donneur regroupées sous forme de blocs. La partie gauche de l'écran correspond à la promesse de don, celle de droite aux opérations funéraires ;
* **Documents** : c'est ici que vous pouvez télécharger/téléverser les documents relatifs au donneur : promesse de don, certificat de décès...
* **Dons financiers** : pour enregistrer des dons financiers destinées au don centre de don ;
* **Cérémonies** : gestion des cérémonies pour ce contact ;
* **Activités** : liste les actions réalisées sur le dossier : impression de documents, envoi de mails, retrait d'effets personnels. C'est ici que vous trouverez les courriels et courriers que vous aurez généré par le système ;
* **Relations** : liste les relations entre contacts, par exemple *personne référente, personne ayant qualité pour pourvoir aux funérailles*... Si la relation est modifiée pour un des contacts, elle l'est aussi dans la fiche du second impliqué dans la relation ;
* **Notes** : contient des notes libres de la part de l'utilisateur et des messages générés par le système, notamment lors des purges ;
* **Tags/Etiquettes** : permet de repérer des contacts particuliers par une étiquette ;
* **Groupes** : liste les groupes auxquels le Contact appartient ;
* **Journal des modifications** : logs pour ce Contact ;
* **Arrivée du Corps** : informations sur le médecin ayant signé le certificat de décès, sur le statut sérologique, les effets personnels du donneur ;
* **Utilisation du Corps** : décrit les utilisations du corps et d'éventuels prélèvements d'organes ou de tissus ;
* **Protocoles in *vivo*** : liste les protocoles dans lesquels le donneur était inclus *de son vivant*.
